# DIT Transfer App v1.0 🚀

![Transfer App Screenshot](screenshots/main-window.png) *Main GUI Dashboard*

> **High-Speed Drive/Folder Transfer Tool**  
> Monitor USB drives, transfer folders/files, multi-target support with resume, hashing & node-sync.

## ✨ Features (Top 10)

| # | Feature | Description |
|---|---------|-------------|
| 1 | **Drive Monitoring** | Auto-detect USB drives & external storage |
| 2 | **Ordner Transfers** | Full folder sync with subdirectories |
| 3 | **Multi-Ziele** | Transfer to multiple destinations simultaneously |
| 4 | **PreHash Verification** | Hash files before transfer for integrity |
| 5 | **Speed Optimization** | Multi-threaded transfers (up to 4GB/s) |
| 6 | **Live Logging** | Real-time progress & error logs |
| 7 | **Resume Capability** | Pause/resume interrupted transfers |
| 8 | **Error Recovery** | Auto-retry failed files with backoff |
| 9 | **Node Sync** | Distributed transfer across network nodes |
| 10 | **Config Profiles** | Save/load transfer configurations |

## 📋 Quick Start

### Simple Usage
```bash
# Windows
build.bat
transfer_app.exe

# Linux
./build.sh
./transfer_app
```

### Config Example (`config.ini`)
```ini
[Settings]
source_path=/mnt/usb
target_path=/home/backup
parallel_threads=4
live_hashing=true
monitor_drives=true
resume_on_interrupt=true
log_level=debug
node_sync_enabled=false
```

## 🛠 Build & Run

### Dependencies
```
Qt 6.5+ (Core, Widgets, Network)
ImGui (Dear ImGui v1.90+)
OpenSSL 3.0+ (for secure node comms)
CMake 3.20+
```

#### Windows (Visual Studio/MSYS2)
```bat
# Using provided build.bat
build.bat

# Manual CMake
mkdir build && cd build
cmake .. -G \"Visual Studio 17 2022\"
cmake --build . --config Release
```

#### Linux (Ubuntu/Debian)
```bash
# Install deps
sudo apt install qt6-base-dev libssl-dev cmake libimgui-dev

# Build
mkdir build && cd build
cmake ..
make -j$(nproc)

# Run
./transfer_app
```

### Testing
```bash
# Unit tests
cd tests/
./run_tests.sh

# Integration test (simulated USB drive)
./test_drive_sync.sh
```

## 🔧 Configuration Examples

### **High-Speed Local Backup**
```ini
[Backup]
source_path=D:\\USB-Drive
target_path=E:\\Backups
parallel_threads=8
pre_hash=true
speed_limit=0  ; Unlimited
```

### **Multi-Node Sync**
```ini
[NodeSync]
source_path=/media/drive
targets=node1:/backup,node2:/raid
node_sync=true
parallel_threads=16
error_retry=5
```
